# PlumbTech E-Commerce Front-End 🛠️

Welcome to **PlumbTech**, a clean, professional, and responsive e-commerce storefront built from scratch. This web application is tailored for showcasing and browsing high-quality plumbing materials, industrial pipes, and water installations.

## 🚀 Live Demo
You can view the live project running here: [Click to View Live Site](https://Oli-via123.github.io/PlumbTech-Website/)

---

## ✨ Features
* **Modern Header & Navbar:** A clean dark-corporate navigation bar with structured menu items and a dynamic layout.
* **Hero Banner Section:** Features a prominent header for "Quality Plumbing Materials" alongside categorized quick-navigation buttons.
* **Call to Action (CTA):** A bold, prominently placed **Shop Now** button anchored to the bottom right of the hero banner.
* **Featured Products Grid:** A clean flexbox card layout showcasing product details, pricing, and "Add to Cart" interactions.

---

## 🛠️ Built With
* **HTML5:** For clean, semantic web layout and page structure.
* **CSS3 (Flexbox & Gradients):** For completely custom styling, fluid grid alignments, interactive transitions, and responsive spacing.
